# Examples

AI Tool CMS 官方示例集合，帮助新开发者在 30 分钟内完成部署与集成。

| 目录 | 说明 |
|------|------|
| [starter/](./starter/) | 快速启动与验证脚本 |
| [docker/](./docker/) | Docker 构建与部署 |
| [plugin/](./plugin/) | 插件开发示例 |
| [api/](./api/) | Public API 调用示例 |

## 快速开始

```bash
chmod +x examples/starter/verify.sh
./examples/starter/verify.sh
pnpm dev:stack
```
