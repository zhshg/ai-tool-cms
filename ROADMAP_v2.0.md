# Roadmap v2.0

**Target:** Platform expansion  
**Theme:** Enterprise & Ecosystem

## 重点功能

### MCP Server GA
- 生产级 MCP 部署（HTTP transport、认证、限流）
- 官方 Cursor / Claude Desktop 插件市场 listing

### Workflow Builder
- 拖拽式工作流编辑器
- 可视化调试与步骤回放

### AI Operator
- 自主运营 Agent：采集调度、内容优化、SEO 修复
- 人工审批节点与策略护栏

### Enterprise Edition
- SSO (SAML / OIDC)
- 审计日志、合规导出
- SLA 与支持工单集成

### Team Workspace
- 多团队、多站点
- 角色与权限细粒度（站点级 RBAC）

### SaaS Cloud
- 托管多租户版本
- 计费与订阅（Stripe）
- 自定义域名与白标

### Mobile App
- iOS / Android 工具浏览 App
- 推送通知（新工具、趋势）

### Browser Extension
- 一键收录当前网页为工具
- 侧边栏对比与收藏

## 架构演进

- GraphQL API（可选）
- 事件溯源审计
- 多区域部署与数据驻留

## 依赖 v1.1

- Plugin Marketplace 基础
- Analytics 数据管道
- API 路径标准化
