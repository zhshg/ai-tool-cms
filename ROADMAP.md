# 路线图

AI Tool CMS 的开发计划与里程碑。

## 阶段一：基础架构

- [ ] Monorepo 工具链（包管理、构建、Lint）
- [ ] 共享包骨架（types、config、logger、utils）
- [ ] 数据库 Schema 与 Prisma 配置
- [ ] Docker 本地开发环境
- [ ] CI/CD 流水线

## 阶段二：核心 API

- [ ] 认证与授权（auth）
- [ ] 工具（Tools）CRUD
- [ ] 分类（Categories）管理
- [ ] 文件上传与存储（storage）
- [ ] 消息队列与 Worker 基础

## 阶段三：Web 站点

- [ ] 工具列表与详情页
- [ ] 分类浏览与搜索
- [ ] 对比、替代品、评测等内容页
- [ ] SEO 优化（seo 包）
- [ ] 多语言与 GEO 支持

## 阶段四：管理后台

- [ ] 内容管理（工具、分类、FAQ、新闻）
- [ ] 定价与发布管理
- [ ] Prompt 与集合管理
- [ ] 审核与工作流

## 阶段五：数据采集

- [ ] 爬虫核心（crawler-core）
- [ ] Crawler 应用与调度（scheduler）
- [ ] 数据清洗与入库流水线
- [ ] 增量更新与去重

## 阶段六：AI 能力

- [ ] AI 内容生成与摘要
- [ ] 自动分类与标签
- [ ] Prompt 模板管理
- [ ] 内容质量评估

## 阶段七：上线与运维

- [ ] 性能优化与缓存
- [ ] 监控、日志与告警
- [ ] 备份与灾备
- [ ] 生产部署文档
