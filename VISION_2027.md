# Vision 2027

**AI Tool CMS — 成为全球领先的 AI 工具内容基础设施**

## 使命

让任何人都能在 30 分钟内搭建、运营并规模化一个世界级的 AI 工具目录。

## 2027 目标

| 维度 | 目标 |
|------|------|
| 开源社区 | 1,000+ GitHub Stars，活跃贡献者生态 |
| 部署规模 | 100+ 自托管生产实例 |
| 内容规模 | 单实例支持 100K+ 工具条目 |
| 开发者 | SDK / MCP 成为 AI 工具数据的事实标准接口之一 |
| 企业 | Enterprise Edition 付费客户与 SaaS 试点 |

## 战略支柱

### 1. AI-Native
内容不是手写出来的，而是 **采集 → AI 增强 → 人工审核 → 自动 SEO** 的流水线产物。2027 年 AI Operator 可自主完成 80% 日常运营。

### 2. Open Platform
REST API、MCP、Webhooks、Plugin 四层开放。第三方可构建垂直行业目录（设计工具、开发工具、医疗 AI）而无需 fork 核心。

### 3. Global by Default
50+ 语言、区域 SEO、本地化工作流。每个市场都有原生体验。

### 4. Observable & Trustworthy
生产级监控、安全合规、透明开源。企业客户可审计每一条 AI 生成内容的来源与版本。

## 核心价值观

- **Open Source First** — 核心永远开源
- **Developer Experience** — 30 分钟上手不是口号
- **Quality over Quantity** — AI 内容必须有质量门禁
